# Getting started with kubeflow pipelines 

1. Follow the tutorial [here](https://www.kubeflow.org/docs/components/pipelines/getting-started/).
2. Extend the demo by:
  - creating 3 to 4 other functions to add to the pipeline
  - or adjusting the functions from the testing labs to work as part of the kubeflow pipeline ( for instance add a data validation step to validate incoming usernames and emails ).
